## 1. 공간 활용 핵심 원칙 (BCG 레퍼런스에서 도출)

### 1.1 "빈 공간 제로" 원칙
BCG 레퍼런스 322p 중 **빈 공간이 의도적으로 남겨진 본문 슬라이드는 0개**. 모든 본문 슬라이드는 콘텐츠가 slot-body의 95%+ 를 채운다.

### 1.2 공간 채우기 전략 (빈 공간 발생 시 BCG가 사용하는 방법)
1. **하단 callout 박스** — 녹색/연두 배경의 시사점 요약 (p15, p20, p06 등)
2. **"Read more" 링크 박스** — 관련 BCG 보고서 링크 (p13, p14, p16, p18, p19)
3. **annotation 텍스트** — 차트 아래 해석 문장 (p07, p08, p09)
4. **하단 산업별/유형별 그리드** — 추가 분류 정보 (p12)
5. **아이콘+설명 박스** — 관련 주제 2~3개 (p06)
6. **기업 사례 로고** — 구체적 기업 사례 (ai-first-win p06)

### 1.3 복합 레이아웃 원칙
BCG는 **단일 레이아웃 유형을 사용하지 않는다**. 하나의 슬라이드에 2~3개의 서로 다른 정보 블록을 결합한다:
- 차트 + 패널 + callout (3블록)
- Big Number + 차트 + 예시 설명 + callout (4블록)
- 테이블 + 링크 + annotation (3블록)
- Ranked + 산업 그리드 + 패널 + 링크 (4블록)

| Insight+Chart | 차트6행 + 패널4문단(16줄) + callout2줄 | **24줄** |
| Ranked | 5항목×(제목+불릿2) + 패널3문단 + 그리드 | **25줄** |

| Table/List | 6행 × (제목+설명2줄) = 18줄 | **18줄** |

### 4.2 필수 하단 채우기 요소
**모든 비면제 슬라이드에 다음 중 최소 1개 필수**:
- callout 박스 (녹색 배경, 시사점 요약)
- annotation 텍스트 (차트 해석)
- "Read more" 링크 박스
- 보조 그리드/테이블

### 4.3 CSS justify-content 재정의
- `space-evenly`만으로는 부족 — **콘텐츠 자체가 충분해야** 함
- justify-content는 보조 수단, **밀도 규칙이 주 수단**
